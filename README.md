### STEP 2: Use Diskpart

Once command line is open, type `diskpart` and press Enter.

Po włączeniu *wiersza poleceń*, wrowadź komendę **`diskpart`** i zatwierdź klawiszem **Enter**

```bash
c:\windows\system32> diskpart
```
